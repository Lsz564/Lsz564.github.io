FAQ

Hoe kan ik een document uitprinten op HZ?

Je kan met je studentenpas aanmelden op de printer en op deze manier uitprinten zolang er saldo op je pas staat

Hoe kan ik een document inscannen en naar mijn eigen laptop versturen?

Je kan dit doen door middel van dezelfde methode als het printen. Echter moet je een andere optie op de printer selecteren om dit te kunnen doen

Hoe kan ik de HZ shop bereiken?

Deze is te bereiken via het Self Service Portal

Hoe kan ik een ruimte boeken?

Dit kan ook via het Self Service Portal
Namelijk onder het kopje "Reservaties"

Parkeerregeling HZ

Rondom de HZ zijn parkeerplekken aanwezig.
Ook bij het PSD-gebouw aan de Prins Hendrikweg (tegenover de hoofdingang van de hoofdlocatie van de HZ aan de Edisonweg) zijn parkeerplekken aanwezig.
Let op: er geldt een parkeerverbod in het gebied tussen de Edisonweg, Oude Veerhavenweg en de Prins Hendrikweg.
Parkeer daarom alleen in de parkeervakken.
